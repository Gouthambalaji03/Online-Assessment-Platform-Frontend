## Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn

### Installation
```bash
git clone <repository-url>
cd Online-Assessment-Platform-Frontend
npm install
```

### Running the Application
```bash
npm start
```

The application will run on `http://localhost:3000`
